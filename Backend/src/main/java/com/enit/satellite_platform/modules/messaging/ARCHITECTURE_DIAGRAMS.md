## Performance Characteristics

### Write Operations
```
Operation                  | Time    | Index Used
---------------------------|---------|---------------------------
Create Message             | 5-10ms  | (insert + index update)
Update Conversation        | 5-10ms  | participants_idx
Mark Message as Read       | 5-10ms  | conversation_timestamp_idx
```

### Read Operations
```
Operation                  | Time    | Index Used
---------------------------|---------|---------------------------
Get Conversations          | < 10ms  | participant_lastmessage_idx
Get Messages (paginated)   | < 10ms  | conversation_timestamp_idx
Count Unread               | < 5ms   | recipient_status_idx
Find Conversation          | < 5ms   | participants_idx (unique)
```

### Data Isolation
```
User A can only:
- See conversations where participants includes A's ID
- Read messages where conversationId matches A's conversations
- Send messages to valid existing users
- Mark messages as read where recipientId = A's ID
```
